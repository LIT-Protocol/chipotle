Overview

This contract is used to manage the configuration of accounts and their associated data.

Mappings using uint256 are always keccak256 of whatever is being mapped. Each struct contains a keccak256 that matches the mapping to prove existence of the struct.

Errors

  • AccountDoesNotExist(uint256 apiKey)
  • GroupDoesNotExist(uint256 apiKey, uint256 groupId)
  • WalletDoesNotExist(uint256 apiKey, uint256 groupId, uint256 Wallet)
  • ActionDoesNotExist(uint256 apiKey, uint256 groupId, uint256 action)
  • UsageApiKeyDoesNotExist(uint256 apiKey, uint256 usageApiKey)
  • InsufficientBalance(uint256 apiKey, uint256 amount)
  • OnlyApiPayerOrPricingOperator(address caller)

Structs

Metadata

Metadata struct for account, group, action, and wallet.

  • uint256 id — keccak256 of the metadata id; used to prove existence of the struct.
  • string name — Name of the metadata.
  • string description — Description of the metadata.

UsageApiKey

UsageApiKey struct for usage API keys.

  • Metadata metadata — Name and description of the usage API key.
  • uint256 apiKeyHash — keccak256 of the base64-encoded API key.
  • uint256 expiration — Expiration date of the API key.
  • uint256 balance — Balance of the API key (funds for payment).

Group

Group struct for groups.

  • Metadata metadata — Name and description of the group.
  • EnumerableSet.UintSet permitted_actions_cid_hash — keccak256 of action IPFS CIDs.
  • EnumerableSet.UintSet Wallets_hash — keccak256 of wallet public keys.
  • bool all_wallets_permitted — Whether all wallets are permitted to use the group.
  • bool all_actions_permitted — Whether all actions are permitted in the group.

Account

Account struct for accounts.

  • Metadata accountMetadata — Name and description of the account.
  • UsageApiKey accountApiKey — API key used to access the account.
  • address creatorWalletAddress — Wallet address of the account creator.
  • EnumerableSet.UintSet usageApiKeysList — Set of usage API key hashes.
  • mapping(uint256 => UsageApiKey) usageApiKeys — Usage key hash → config.
  • EnumerableSet.UintSet groupList — List of group IDs.
  • mapping(uint256 => Group) groups — Group ID → group config.
  • EnumerableSet.UintSet actionHashesList — Set of action hashes.
  • mapping(uint256 => Metadata) actionMetadata — Action CID hash → metadata.
  • bool managed — Whether the LIT-node can help manage the key.
  • mapping(uint256 => uint256) walletAddressHash — Wallet address hash → index.
  • mapping(uint256 => uint256) wallet_derivation — Wallet address hash → derivation path.
  • mapping(uint256 => Metadata) walletDerivationMetadata — Wallet hash → metadata.
  • uint256 walletCount, actionCount, groupCount — Counters.

State Variables

  • mapping(uint256 => Account) accounts — API key hash → account config.
  • mapping(uint256 => uint256) allApiKeyHashes — Any API key hash → master account API key hash.
  • mapping(uint256 => uint256) allWalletAddressHashes — Counter → wallet address hash.
  • mapping(uint256 => uint256) pricing — Pricing item ID → price.
  • address public api_payer — Pays for state mutations from API calls; may mutate on behalf of key holder.
  • address public pricing_operator — May mutate pricing and related state.
  • uint256 public nextWalletCount — Counter for globally unique wallet hashes.

Internal / private helpers

These revert or return existence checks; they are used by the public API and are documented in the contract NatSpec:

  • usageApiKeyExists — Returns whether a usage API key exists (reverts if account not mutable by caller).
  • groupExists — Returns whether a group exists for the account.
  • actionExists — Returns whether an action is permitted in the given group.
  • WalletExists — Returns whether a wallet is in the group's permitted set.
  • revertIfAccountDoesNotExistAndIsMutable — Reverts with AccountDoesNotExist if account missing or immutable by caller.
  • revertIfGroupDoesNotExist, revertIfActionDoesNotExist, revertIfWalletDoesNotExist, revertIfUsageApiKeyDoesNotExist — Revert with the corresponding error if the entity does not exist.
