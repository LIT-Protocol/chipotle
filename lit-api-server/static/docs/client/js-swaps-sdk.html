SDK / Swap API

JavaScript Swap SDK

Wrapper for the swap intents API: token list, quote requests, fill/accept quote, swap status, open requests and quotes, quote balances, and attempt swap. Use swaps_sdk.js when the server mounts swap routes at /swaps/v1/.

Tip Types match abstractions/intents/swaps/models.rs and endpoints.rs. All methods return Promises and throw on non-2xx HTTP responses. Request bodies use snake_case (e.g. origin_chain, swap_request_id).

Prerequisites

Note
  • baseUrl — API server URL (default http://localhost:8000). Routes are requested at baseUrl/swaps/v1/.
  • Swap routes — Server must mount swap endpoints (token_list, new_quote_request, fill_quote, get_open_swap_requests, get_open_quotes, get_quote_balances, attempt_swap_request, etc.).
  • Chain keys — Use lowercase chain identifiers (e.g. ethereum, yellowstone) for origin/destination chain in quote requests.

Quick Example

import { createSwapsClient, QUOTE_PRICING_ORIGIN } from './swaps_sdk.js';

const client = createSwapsClient('http://localhost:8000');

const contractAddress = await client.getContractAddress();
const { swap_requests } = await client.getOpenSwapRequests();
const { quotes } = await client.getOpenQuotes();

const { swap_request_id, transaction_hash } = await client.newQuoteRequest({
  from: '0x...',
  originChain: 'ethereum',
  originSymbol: 'ETH',
  originAmount: 1e18,
  destinationChain: 'yellowstone',
  destinationSymbol: 'LIT',
  destinationAmount: 1000,
  slippage: 1,
  pricingType: QUOTE_PRICING_ORIGIN,
  quoteDeadlineSeconds: 60,
  originAddress: '0x...',
  refundAddress: '0x...',
  transactionDeadlineSeconds: 0,
});

const { quote_id } = await client.fillQuote({
  swapRequestId: swap_request_id,
  quoteDeadlineSeconds: 60,
  providerRefundAddress: '0x...',
});

const balances = await client.getQuoteBalances(quote_id);
await client.attemptSwapRequest(quote_id);

Client & factory

createSwapsClient(baseUrl?)
Factory that returns a LitSwapsApiClient. Default base URL: http://localhost:8000.
Export: createSwapsClient
new LitSwapsApiClient({ baseUrl })
Construct swaps client with optional baseUrl. Trailing slashes are stripped. Requests go to baseUrl/swaps/v1/.

Pricing constants

Use these when building a new quote request:

  • QUOTE_PRICING_ORIGIN = 0 — Fees taken from origin amount (destination amount fixed).
  • QUOTE_PRICING_DESTINATION = 1 — Fees taken from destination amount (origin fixed).

Token list

getTokenList()
Returns the list of supported tokens for the swap UI.
GET /token_list

Returns: Promise<TokenListResponse> — { tokens: TokenListItem[] }

New quote request

newQuoteRequest(options)
Creates a new swap request on-chain and returns swap_request_id and transaction_hash. Options use camelCase; the SDK sends snake_case to the API.
POST /new_quote_request

Options (key fields):

PropertyTypeDescription
fromstringMessage sender (hex address)
originChainstringOrigin chain key (e.g. "ethereum", "yellowstone")
originSymbolstringOrigin token symbol
originAmountnumberOrigin amount (converted to number for JSON)
destinationChainstringDestination chain key
destinationSymbolstringDestination token symbol
destinationAmountnumberDestination amount
slippagenumberAcceptable slippage
pricingTypenumber0 = Origin, 1 = Destination (use QUOTE_PRICING_ORIGIN / QUOTE_PRICING_DESTINATION)
quoteDeadlineSecondsnumberHow long to wait for a quote (0–60)
originAddressstringOrigin address (hex)
refundAddressstringRefund address if tx fails (hex)
transactionDeadlineSecondsnumberOptional; 0 if not set
messagestringOptional message

Returns: Promise<NewSwapResponse> — { swap_request_id, transaction_hash, swap_request_expiry, fees, signed_input_hash }

Fill quote

fillQuote(options)
Solver fills a quote for a swap request (e.g. calls contract newQuote). Returns quote_id and transaction hash.
POST /fill_quote

Options: swapRequestId (string), quoteDeadlineSeconds (number), providerRefundAddress (string, hex), message (string, optional).

Returns: Promise<FillQuoteResponse> — { quote_id, transaction_hash, pkp_address, swap_request_id, quote_expiry, fees, total_fees, signed_input_hash }

Accept quote

acceptQuote(options)
User accepts a quote; returns the PKP address to send funds to.
POST /accept_quote

Options: quoteId (string).

Returns: Promise<AcceptQuoteResponse> — { pkp_address }

Swap status

getSwapStatus(quoteId)
Gets the status of a swap by quote id.
GET /get_swap_status/<quote_id>

Returns: Promise<GetSwapStatusResponse> — { state, details? }. state is one of: Pending, Processing, Success, Expired, Refunded, Other.

Contract address & open lists

getContractAddress()
Returns the quote storage contract address (hex string). Use this when the front end needs to call the contract (e.g. MetaMask newQuote).
GET /get_contract_address

Returns: Promise<string>

getOpenSwapRequests()
Returns open swap requests from the contract (via getRecentSwapRequests).
GET /get_open_swap_requests

Returns: Promise<GetOpenSwapRequestsResponse> — { swap_requests: SwapRequestData[] }

getOpenQuotes()
Returns open quotes from the contract (via getRecentQuotes + linked swap request data).
GET /get_open_quotes

Returns: Promise<GetOpenQuotesResponse> — { quotes: QuoteData[] }. Each QuoteData includes swap_request_data (the linked swap request).

Quote balances

getQuoteBalances(quoteId)
Fetches quote and swap request by quote id, then returns PKP address, source/destination chain, balances in wei on both chains, and whether each balance is sufficient for the swap amounts.
GET /get_quote_balances/<quote_id>

Parameters: quoteId (string — decimal, e.g. from fillQuote response).

Returns: Promise<QuoteBalancesResponse> — { pkp_address, source_chain, destination_chain, source_balance_wei, destination_balance_wei, source_balance_sufficient, destination_balance_sufficient }

Attempt swap

attemptSwapRequest(quoteId)
Attempts to execute the swap for the given quote (sends origin and destination chain transfers). Fails with 402 Payment Required if PKP balance is insufficient on either chain.
GET /attempt_swap_request/<quote_id>

Parameters: quoteId (string).

Returns: Promise<string> — Empty string on success.

Throws: On non-OK response (e.g. 402 Payment Required).
