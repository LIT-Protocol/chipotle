## Welcome

Welcome to the Lit Actions SDK Docs.  These functions can be used inside a Lit Action.  You should prefix each function with "Lit.Actions." so to call "encrypt()" you should do "Lit.Actions.encrypt()"  To understand how these functions fit together, please view the documentation for this SDK, located at [https://developer.litprotocol.com/](https://developer.litprotocol.com/)

## Encryption

Encryption and decryption functions.  These functions are used to encrypt and decrypt data.  The data is encrypted & decrypted using a symmetric key derived from the secret key of the PKP.
